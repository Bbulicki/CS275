This webpages provides user with hourly weather forecast of the desired location, provided that the user has an authorized key.
To see the hourly weather forecast, the user can enter their key in the text field provided and then click on the submit key button.
If the key is correct and the zipcode is acceptable, hourly weather forecast of the current location will be displayed in a table below.
If the key is incorrect, a message will appear saying that the page could not be reached.
